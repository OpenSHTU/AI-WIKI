# Start Here

This page is the English learning route for the AI knowledge base. The English section currently provides bilingual navigation and overview-level placeholders, while the Chinese section keeps the deeper note graph.

## Recommended Path

1. [Math Foundations](./foundations/math/): vectors, matrices, gradients, probability, and information theory.
2. [Machine Learning](./foundations/ml/): losses, optimization, backpropagation, and generalization.
3. [NLP](./nlp/): text processing, language modeling, representation, and sequence modeling.
4. [LLM](./llm/): Transformer, pretraining, alignment, RAG, agents, and inference.
5. [Multimodal AI](./multimodal/): vision-language models, CLIP, diffusion, and alignment.
6. [World Models](./world-models/): environment modeling, state spaces, and planning.
7. [Embodied AI](./embodied-ai/): perception-action loops, reinforcement learning, and robotics.
