# 从这里开始

这是一条面向 AI 知识库的主学习路线。当前阶段只搭建框架，因此每个节点会先指向对应占位笔记。

## 推荐路线

1. [数学基础](./foundations/math/)：先掌握向量、矩阵、梯度、概率和信息论。
2. [机器学习基础](./foundations/ml/)：理解损失函数、优化、反向传播和泛化。
3. [NLP 基础](./nlp/)：从文本处理、语言模型、词表示走向序列建模。
4. [Transformer 与 LLM](./llm/)：学习自注意力、预训练、对齐、RAG 和 Agent。
5. [多模态](./multimodal/)：理解视觉语言模型、CLIP、扩散模型与跨模态对齐。
6. [世界模型](./world-models/)：进入环境建模、状态空间和规划。
7. [具身智能](./embodied-ai/)：把感知、决策、控制与机器人系统连接起来。
